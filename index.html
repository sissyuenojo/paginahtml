Programação de uma página em HTML

Para programar uma página de um site, precisamos aprender usar algumas tags.

Essas são as tags usadas para fazer o cabeçalho da página: h# /h Precisamos lembrar que elas devem ser abertas (< h# >) e fechadas (< /h# >). Não podemos esquecer do fechamento das tags, que são os sinais de maior e menor (< >). No fechamento da tag ainda acrescentamos a barra (/)..
